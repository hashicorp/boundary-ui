# LinkButton

An achor element component styled to mimic `Rose::Button` component.
Styles supported: primary, secondary, warning, and ghost.
Disabling link button is not supported as it is an action on form components only.

```html
<Rose::LinkButton
  @route="index"
  @style="primary"
  @iconRight="flight-icons/svg/entry-point-16"
>
  Content
</Rose::LinkButton>
```

Similar to `Rose::Button`, icon only link button is supported.

```html
<Rose::LinkButton
  @route="index"
  @style="primary"
  @iconOnly="flight-icons/svg/help-16"
/>
```
