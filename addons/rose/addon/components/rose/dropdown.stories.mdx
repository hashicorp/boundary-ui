# Dropdown

A dropdown component wrapped on top of ember-basic-dropdown.

```htmlbars
<Rose::Dropdown @label="Dropdown" as |dropdown|>
  <dropdown.option>Menu Item</dropdown.option>
  <dropdown.option>Menu Item</dropdown.option>
  <dropdown.option>Menu Item</dropdown.option>
</Rose::Dropdown>
```

Dropbowns can just be icons, however, icons need to be selected from an existing list. Currently, only chevron and user are available.
