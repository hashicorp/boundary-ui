# Card Link

Card link component wraps content with `LinkTo` component. Supports title and description.

```html
<Rose::Card::Link
  @title="Card title"
  @description="Card description"
  @id="resource_id_1234"
  @icon="flight-icons/svg/org-16"
  @route="about"
/>
```
