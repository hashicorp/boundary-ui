# Card Link

Card link component wraps content with `LinkTo` component. Supports title and description.

```handlebars
<Rose::Card::Link
  @title='Card title'
  @description='Card description'
  @id='resource_id_1234'
  @icon='flight-icons/svg/org-16'
  @route='about'
/>
```

export const Template = (args) => ({
  template: hbs`
      <Rose::Card::Link
        @title={{title}}
        @description={{description}}
        @id={{id}}
        @icon={{icon}}
      />
    `,
  context: args,
});
