# Generic Navigation Link

Proxies the built-in `LinkTo` component, applying a class `rose-nav-link`.

```handlebars
<Rose::Nav::Link @route='index'>Text</Rose::Nav::Link>
```

export const Template = (args) => ({
  template: hbs`
      <Rose::Nav::Link @route='index'>Text</Rose::Nav::Link>`,
  context: args,
});
