# Rose/Form/Radio/Card (Deprecated)

Rose/Form/Radio/Card has been deprecated in favor of [HDS::Form::RadioCard](https://design-system-components-hashicorp.vercel.app/components/form/radio-card)

### <Rose::Form::Radio::Card> usage:

# Form Radiofield in a wide format

Renders a form radiofield in a Wide format.

```handlebars
<Rose::Form::Radio::Card
  @label='Yes'
  @error={{false}}
  @value='yes-agree'
  @selectedValue='yes-agree'
  @disabled={{false}}
  @changed={{fn}}
  @layout='wide'
/>
```

Renders a form radiofield in a tile format.

```handlebars
<Rose::Form::Radio::Card
  @label='Yes'
  @error={{false}}
  @value='yes-agree'
  @selectedValue='yes-agree'
  @disabled={{false}}
  @changed={{fn}}
  @layout='tile'
/>
```

export const Template = (args) => ({
