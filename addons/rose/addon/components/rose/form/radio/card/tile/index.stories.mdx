# Form RadioCard field in tile format

Renders a form radioCard field in a tile format.

```htmlbars
<Rose::Form::Radio::Card::Tile
  @label="Yes"
  @error={{false}}
  @value="yes-agree"
  @selectedValue="yes-agree"
  @disabled={{false}}
  @changed={{fn}}
  />
```
