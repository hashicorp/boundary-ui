# Form checkboxfield group

Renders a form checkboxfield group.

```htmlbars
<Rose::Form::Checkbox::Group
  @disabled={{false}}
  @name={{name}}
  @selectedValue="disagree"
  @changed={{fn}}
  @inline={{false}}
  as |checkboxGroup|
>
  <checkboxGroup.checkbox
    @label="Agree"
    @value="agree"
  />
  <checkboxGroup.checkbox
    @label="disagree"
    @value="disagree"
  />
  <checkboxGroup.checkbox
    @label="maybe"
    @value="maybe"
  />
</Rose::Form::Checkbox::Group>
```
