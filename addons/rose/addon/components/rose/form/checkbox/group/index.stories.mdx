# Form checkboxfield group

Renders a form checkboxfield group.

```htmlbars
<Rose::Form::Checkbox::Group
  @name='checkbox-group'
  @items={{this.items}}
  @selectedItems={{this.selectedItems}}
  @onChange={{fn this.onChange}}
  as |group|
>
  <group.checkbox
    @label={{group.item.name}}
    value={{group.item.id}} />
</Rose::Form::Checkbox::Group>
```
