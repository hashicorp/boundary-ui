# Form Textarea

Renders a simple form textarea.

```handlebars
<Rose::Form::Textarea
  @name='field'
  @value='Descriptive content'
  @label='Description'
  @helperText='Enter a description'
  @error={{false}}
  @disabled={{false}}
  readonly={{false}}
  @link='www.example.net'
  @linkText='link text goes here'
/>
```
