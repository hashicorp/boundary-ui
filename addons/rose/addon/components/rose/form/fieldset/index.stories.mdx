# Form Fieldset

Renders a simple fieldset with title, description and body and an optional helperText

```html
<Rose::Form::Fieldset @helperText={{helperText}}>
  <:title>Title</:title>
  <:description>Description</:description>
  <:body>Rest of the form</:body>
</Rose::Form::Fieldset>
```
