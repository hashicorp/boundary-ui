# Form Helper Text

Displays helpful messages related to a form field. This component is used
by form fields and should not be used directly.

```handlebars
<Rose::Form::HelperText
  @id='helper-text-{{field.id}}'
  @link='link'
  @linkText='link text goes here'
>
  Helper text
</Rose::Form::HelperText>
```
