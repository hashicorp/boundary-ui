# Form Checkbox

Renders a form input checkbox.

```htmlbars
<Rose::Form::Checkbox
  @id="form-field-1"
  name="boolean-value"
  @label="Yes?"
  @checked={{false}}
  @error={{false}}
  disabled={{false}}
  />
```
