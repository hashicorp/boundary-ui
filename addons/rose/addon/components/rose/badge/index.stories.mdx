# HDS Badge (Recommended)

> **_NOTE:_** Replaces the deprecated Rose/Badge.

```handlebars
<Hds::Badge @text='Success badge' @color='neutral' />
```

Refer to the [HDS Badge documentation](https://design-system-components-hashicorp.vercel.app/components/badge) for a complete overview

export const HDSTemplate = (args) => ({
  template: hbs`<Hds::Badge @text={{text}}
      @color={{color}}
      />`,
  context: args,
});
