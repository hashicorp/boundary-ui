# Toolbar

Renders a toolbar. Items may be added directly into the toolbar, or within
predefined "regions" for left and right alignment, respectively.

```handlebars
<Rose::Toolbar as |toolbar|>
  // place items here
  <toolbar.primary>
    // place left-aligned items here
  </toolbar.primary>
  <toolbar.secondary>
    // place right-aligned items here
  </toolbar.secondary>
</Rose::Toolbar>
```

export const Template = (args) => ({
  template: hbs`
      <Rose::Toolbar as |toolbar|>
        toolbar
      </Rose::Toolbar>`,
  context: args,
});
