# MetadataList

A page section component that allows for a list of metadata items.

```handlebars
<Rose::MetadataList @orientation='vertical' as |list|>
  <list.Item @icon='arrow-up' @color='success'>
    Text Here
  </list.Item>
</Rose::MetadataList>
```
