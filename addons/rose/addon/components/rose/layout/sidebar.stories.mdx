# Layout|Sidebar

A top-level layout/positioning component that allows two distinct sections to
be configured: header and body.

```htmlbars
<Rose::Layout::Sidebar as |layout|>
  <layout.sidebar>
    sidebar content
  </layout.sidebar>
  <layout.body>
    body content
  </layout.body>
</Rose::Layout::Sidebar>
```
