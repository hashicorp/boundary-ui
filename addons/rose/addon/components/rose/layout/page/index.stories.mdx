# Layout|Page

A page layout/positioning component that allows five distinct sections to be configured: breadcrumbs, header, actions, navigation, and body.

```handlebars
<Rose::Layout::Page as |page|>
  <page.breadcrumbs />
  <page.header />
  <page.actions />
  <page.navigation />
  <page.body />
</Rose::Layout::Page>
```
