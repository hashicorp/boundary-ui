# Layout|BodyContent

A page layout/positioning component that allows two distinct sections to be configured: body and sidebar.

```handlebars
<Rose::Layout::BodyContent as |bc|>
  <bc.Body />
  <bc.Sidebar />
</Rose::Layout::BodyContent>
```
