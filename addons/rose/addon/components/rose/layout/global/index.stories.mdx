# Layout|Global

A top-level layout/positioning component that allows three distinct sections to
be configured: header, body, and footer.

```handlebars
<Rose::Layout::Global as |layout|>
  <layout.header />
  <layout.body />
  <layout.footer />
</Rose::Layout::Global>
```
