# LoadingButton

A simple LoadingButton component with two states - loading and reloading that renders an HTML `<button>.` 
element built by extending the existing Rose|Button component.

```htmlbars
<Rose::LoadingButton>
  loading button
</Rose::LoadingButton>
```
