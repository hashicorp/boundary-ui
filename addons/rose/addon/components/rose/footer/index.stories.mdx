# Footer

A footer component that allows four distinct sections to be configured: brand, nav, and text.

```handlebars
<Rose::Footer as |footer|>
  <footer.nav as |nav|>
    <nav.link @route='index'>Section</nav.link>
    <nav.link @href='index'>Section</nav.link>
  </footer.nav>
  <footer.brand @logo='logo' @text='Product Name' />
  <footer.text>Text</footer.text>
</Rose::Footer>
```

export const Template = (args) => ({
  template: hbs`
      <Rose::Footer as |footer|>
        <footer.nav as |nav|>
          <nav.link @route="index">Section</nav.link>
          <nav.link @href="index">Section</nav.link>
        </footer.nav>
        <footer.brand @logo="logo" @text={{titleText}} />
        <footer.text>{{footerText}}</footer.text>
      </Rose::Footer>`,
  context: args,
});
